# Deadlock AI-Assistant

## Usage

1. Install UV from https://docs.astral.sh/uv/getting-started/installation/
2. Install dependencies
    ```bash
    uv sync
    ```
3. Install pre-commit hook
    ```bash
    pre-commit install
    ```
4. Run the main.py
    ```bash
    uv run python3 main.py
    ```
